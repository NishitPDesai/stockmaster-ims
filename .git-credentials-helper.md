## Option 2: Use Git Credential Manager

Windows Credential Manager will prompt you:
```bash
git push origin main
# Enter username: desainishit1111@gmail.com
# Enter password: <your personal access token>
```

## Option 3: Update Remote URL with Token

```bash
git remote set-url origin https://<TOKEN>@github.com/NishitPDesai/stockmaster-ims.git
```
